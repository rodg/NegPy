# Keyboard Shortcuts

Summary of available keyboard shortcuts in NegPy.

## Navigation
| Key | Action |
|-----|--------|
| `Left Arrow` | Previous image |
| `Right Arrow` | Next image |

## Image Adjustments (High Speed)
| Key | Action |
|-----|--------|
| `Q` / `A` | Increase / Decrease **Density** (0.01) |
| `W` / `S` | Increase / Decrease **Grade** (0.01) |
| `E` / `D` | Increase / Decrease **Magenta** (0.01) |
| `R` / `F` | Increase / Decrease **Yellow** (0.01) |
| `X` / `Z` | Increase / Decrease **Crop Offset** (1.0) |

## Tools
| Key | Action |
|-----|--------|
| `Shift + W` | Toggle White Balance Picker |
| `Shift + C` | Toggle Manual Crop Tool |
| `Shift + D` | Toggle Dust Spot Picker |

## Geometry & Orientation
| Key | Action |
|-----|--------|
| `[` | Rotate 90° CCW |
| `]` | Rotate 90° CW |
| `H` | Flip Horizontal |
| `V` | Flip Vertical |

## System Actions
| Key | Action |
|-----|--------|
| `Ctrl + E` | Export current image |
| `Ctrl + C` | Copy settings from current image |
| `Ctrl + V` | Paste settings to current image |
